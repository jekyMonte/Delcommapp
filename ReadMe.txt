delcommapp
Adalah aplikasi yang dapat digunakan untuk menghapus command pada capture atau text pada file txt.

Cara penggunaan:
1. Buka file delcommapp.exe
2. Upload file yang ingin di generate
3. Silahkan masukan command atau text dimana awal text yang ingin di hapus, contoh "show tech"
4. Setelah itu silahkan klik Generate
6. Jika statusnya sudah Done silahkan klik Download Files untuk menyimpan hasil generate

Jika masih ada yang ingin di hapus tinggal masukan kembali apa yang di hapus, karena proses generate per command

Jika ada masalah atau ada yang bingung, silahkan kirim email ke netoprmgr@compnet.co.id
